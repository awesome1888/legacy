<Container>

На нашем сайте не производится целенаправленный сбор персональных данных. Форма отправки сообщения содержит поле для указания способа связи с автором, однако оно является необязательным для заполнения.

Любые персональные данные, которые автор добровольно указал в тексте сообщения, либо в поле для способа связи, нами не обрабатываются и не хранятся. Сообщения после прочтения немедленно удаляются.

Если у вас есть другие вопросы, пожалуйста, свяжитесь с нами по адресу <Email />.

</Container>
